solution of NC218034

2021-03-22 思维 ， OI

statement

给定一个串 sss ，和一个指令串 TTT ，长度相同。某个神必壬在输入时遵循以下的原则：输入当前的 sis_isi​ 后， 若 Ti=LT_i=\texttt{L}Ti​=L ，则下一次输入在 iii 的左边；否则在 iii 的右边。问按此规则输入完后的串是啥样的。

solution

乍一看似乎是个链表。但其实不需要。

注意到每次的 TiT_iTi​ 实质上会将整个输入的串分成两个部分：在当前的位置左边的（即前缀），和在当前位置右边的（即后缀）。

我们可以来个讨论：

  • 若 Ti=LT_i=\texttt LTi​=L ，那么 sis_isi​ 在输入完后一定是从当前字母左边开始的后缀的一部分；
  • 同理，若 Ti=RT_i=\texttt RTi​=R ，那么 sis_isi​ 在输入完后一定是从当前字母右边开始的前缀的一部分。

因此，我们只需要根据 TiT_iTi​ ，分别记录最后答案的后缀 xxx 与前缀 yyy ，最后拼在一块就好了。

#include <iostream>
#include <cstdio>
#include <algorithm>
#include <string>

using namespace std;

string a, b, x, y; // x 后缀，y 前缀

int main()
{
	cin >> a >> b;
	int n = a.length();
	for (int i = 0; i < n; i++)
	{
		if (b[i] == 'L') x += a[i]; 
		else y += a[i]; 
	}
	reverse(x.begin(), x.end()); 
	cout << y << x << "\n";
	return 0;
}
